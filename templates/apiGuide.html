How to use API


This site's API will return data on all Pokemon when the request is made

The data is taken from Kaggle, the link for this is below

You can get the data for a single Pokemon by going to the link to data for all Pokemon and adding '/[number]' to the end of the URL, with the number being the Pokedex ID of that Pokemon

E.G. for Charizard, add '/6' to the end of the URL for all Pokemon
